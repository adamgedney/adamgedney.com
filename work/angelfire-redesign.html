Angelfire.com Redesign

A Full Sail Univ. class project redesign of angelfire.com

Project Description

Angelfire.com was a monumental website in the 1990's. Many people hosted websites and email accounts through their services. For a Full Sail class project, I redesigned their website and interface. Not all links on this project are functional. It was more an exercise in responsive design and site planning.
