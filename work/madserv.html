Madserv.us

An authentication API for client login & centralized user registration

Project Description

Madserv.us was a wonderfully challenging project. I created an authentication API much like OAuth. It uses a secret key and a token, as well as 64bit encryption to secure user data as it passes across multiple domains. If you take a look at the API page or screenshot below, you'll get a sense for how the secure authentication process takes place. This API wasn't designed for commercial use, but only as a Full Sail class project. The other students built eCommerce sites that used my API to authenticate their users.

https://github.com/adamgedney/MadServ
