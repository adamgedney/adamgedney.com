Gimme Sauce

Gimme Sauce

A Pinterest style affiliate product marketing website

Project Description

Gimmesauce.com was an experiment in using node.js and Mongo Database. I used an hybrid of a MEAN stack on the server. i'm running Linux, Mongo DB, Express framework, and node.js. I am not using Angular, but for ease I'll call it a MEAN stack. The site allows the admin to upload an image, title, description, and product link. All the products link to an Amazon affiliate account for monetization. Only the strangest or coolest products make the list!

https://github.com/adamgedney/gimmesauce
