Jericho Canyon Vineyard

Jericho Canyon Vineyard

A website for a Napa Valley Vineyard

Project Description

Jerichocanyonvineyard.com was one of my first large scale projects. Built on Wordpress, with a custom theme, it's a 100+ page eCommerce site & blog utilizing SSL for secure transactions. Wine club members have the ability to log in and receive special discounts based on their membership tier. The site has since been taken over by company staff, but the inital design and front-end development were mine. Not the prettiest site by today's standards, but it was this project, back in 2012, that rekindled my passion for web.

This project is not on github.
