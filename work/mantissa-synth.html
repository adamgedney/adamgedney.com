Mantissa

Mantissa Software Synth

A software synthesizer Adobe AIR app

Project Description

The Mantissa Software Synth was a neat little Flash/Actionscript project. The Mantissa is a software synthesizer I designed. It generates sound by manipulating the bitstream in Actionscript. I run the bitstream through multiple filters to produce different effects based on UI knob triggers. I designed all the assets in Flash. After debugging was finished I converted the project to an Adobe AIR application. I discovered that Flash is incapable of flawlessly handling sound generation and generation is better suited to more core languages like C.
