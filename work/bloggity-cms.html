Bloggity

Bloggity.org

A blog/CMS website

Project Description

Bloggity.org was a super fast project for one of my Full Sail classes. Bloggity is a CMS blog site built on PHP, with front end development done in Javascript. In just a week I comped, then coded the interface, and added the server side PHP programming to interact with a MySQL database. I also added an XML generator in PHP and provided an RSS API for getting blog posts. It was a fun little project.
