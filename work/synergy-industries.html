Synergy Industries

A website for a NY based event production company

Project Description

Synergyindustries.org is a single page scrolling static website built for a NY event production agency. The website showcases and links to artists they've booked, as well as venues they hosted events in. A clean, minimal interface that promotes readability was the goal on theis project.

https://github.com/adamgedney/synergy
