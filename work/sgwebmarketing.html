SG Web Marketing

A website for a FL based SEO company

Project Description

Sgwebmarketing.com is a mulit-page website designed to be content driven and search engine optimized. This brochure site was designed to be clean, minimal, and usable. It grants easy access to the company services and their embedded Wordpress blog. The only server-side programming was done on the contact forms. Simple PHP mail controllers were built to handle emailing the company.

https://github.com/adamgedney/sginternetmarketing
