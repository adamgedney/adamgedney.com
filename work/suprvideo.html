Suprvideo

A ffmpeg based video conversion web app

Project Description

Suprvideo.com is a favorite project of mine. In PHP I wrote terminal scripts to run ffmpeg video conversion commands. The suprvideo site allows the user to upload a video then have it converted to other video formats as well as have its audio track ripped to mp3. The user can then download any of the video or audio files. It converts from almost any format to webm, mp3, mp4, mov, flv, and ogv.

https://github.com/adamgedney/suprvideo
